# deepin-pdfium

development library for pdf on Deepin.

### Build dependencies

- qtbase5-dev
- chardet
-  lcms2 
- freetype2 
- libopenjp2

### Build from source code

```
$ cd /path/to/your/source/code
$ mkdir build
$ cd build
$ qmake ../
$ make
$ sudo make install
```
